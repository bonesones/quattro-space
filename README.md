# Quattro Space Frontend

Базовый шаблон проекта на Vite (vanilla JS) с Tailwind CSS для дальнейшей натяжки на MODX.

## Скрипты

- `npm install` — установка зависимостей
- `npm run dev` — запуск dev-сервера Vite
- `npm run build` — сборка production-билда в папку `dist`
- `npm run preview` — предпросмотр собранного билда

## Структура

- `index.html` — базовый шаблон страницы
- `src/main.js` — точка входа JS
- `src/style.css` — подключение Tailwind и глобальные стили
- `tailwind.config.mjs` — конфиг Tailwind (пути к шаблонам)
- `postcss.config.mjs` — конфиг PostCSS
